## flex

```html
<style>
    .container {
      height: 100vh;
      border: 10px solid royalblue;
      /* 
      1. flex 선언
      flex container : 감싸고 있는 부모 요소,
      flex items : 구성하고 있는 아이템 요소
      */
      display: flex;
      /*
      2. flex 방향설정
      item들이 쌓이는 방향을 설정할 수 있음.
      row (기본값), column, row-reverse ,column-reverse
      */
      flex-direction: column;
      /*
      3. container 내부에 item 배치(wrap)
      no-wrap (기본값), wrap, wrap-reverse*/
      flex-wrap: wrap;
    }
  </style>
```

연습을 통해서 어떻게 사용해야하는지 파악한다.

column은 90도를 꺽어서 사용한다고 생각을 하는게 편하다.

## flex_정렬

```html
<style>
		.container {
			height: 100vh;
      border: 10px solid royalblue;
      display: flex;
      /* 2. justify-content (x축)
        flex-direction이 row일 때, 가로 정렬
                        column 일 때, 세로 정렬
        flex-start(왼쪽 - 방향시작)
        center(가운데)
        flex-end(오른쪽 - 방향의 끝)
        space-between(사이에 여백)
        space-around(item 좌우 여백 동일)
        space-evenly(item 및 외곽 여백 동일)
      */
      flex-direction: row;
      justify-content: center;
      /* 3. align-items (y축)
        flex-start(시작점)
        center(가운데)
        flex-end(끝)
        baseline : 폰트를 바꿔서 확인!
      */
      align-items: baseline;
    }
    .item {
      height: 100px;
      width: 100px;
      font-size: 1rem;
    }
    /* .item2 {
      font-size: 2rem;
    }
    .item3 {
      font-size: 1rem;
    } */
    /* 
    1. flex-grow
    남은 여백을 나눠 가진다.
    기본 값은 0.
    */
    /* .item2 {
      flex-grow: 2;
    }
    .item3 {
      flex-grow: 3;
    } */
	</style>
```

1. flex-grow
   * 남은 여백을 나눠 가진다.
   * 기본 값은 0.

2. justify-content (x축)
   * flex-direction이 row일 때, 가로 정렬
                     column 일 때, 세로 정렬
   * flex-start(왼쪽 - 방향시작)
   * center(가운데)
   * flex-end(오른쪽 - 방향의 끝)
   * space-between(사이에 여백)
   * space-around(item 좌우 여백 동일)
   * space-evenly(item 및 외곽 여백 동일)
3. align-items (y축)
   * flex-start(시작점)
   * center(가운데)
   * flex-end(끝)
   * baseline : 폰트를 바꿔서 확인!

4. self

   * 개인적으로 바꾸는 것은 self를 이용

     ex) align-self : flex-end

   * order --> 기본적으로는 모두 0이다. 우선순위 두기 위해서는 -1을 이용한다.
