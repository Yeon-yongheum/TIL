## 참조

### 인라인 참조

```html
<h1 id="green" style="color: purple">보라색</h1>
```

### 내부 참조

```html
<head>
  <!-- 내부참조(embeded) head 태그 내에 style 태그 -->
  <style>
    /* 태그 선택자 */
    h1 {
      color: red;
    }
    h3 {
      color: pink;
    }
    /* 클래스 선택자 : .클래스명 */
    .blue {
      color: blue;
    }
    .brown {
      color: brown !important;
    }
    /* 아이디 선택자 : #아이디명 */
    #green {
      color: green;
    }
  </style>
</head>
```

* 우선 순위는 아이디 > 클래스 > 태그

### link 참조

```html
 <!-- 외부파일 링크-->
  <link rel="stylesheet" href="01_style.css">
```

이런식으로 불러서 사용

## UNIT

```html
<body>
  <h1>단위를 알아봅시다.</h1>
  <!-- rem
    root 요소의 배수
    html : 16px (브라우저 기본)
    ol : 2rem -> 32px (16*2)
    li : 2rem -> 32px (16*2)
  -->
  <ol>
    ol_2rem 32
    <li>
      ol li_2rem 32
    </li>
  </ol>
  <!-- em
    상위 요소의 배수
    html : 16px 
    ul : 2em -> 32px
    li : 원래 ul 밑에 있어서 32px
         2em -> 32px*2 -> 64px
  -->
  <ul>
    ul_2em 32
    <li>
      ul li_2em 64
    </li>
  </ul>
  <!-- vw, vh -->
  <span class="vw">10vw</span>
  <span class="vh">10vh</span>
  <div class="div-vw"></div>
  <div class="div-vh"></div>
  <!-- vmin -->
  <div class="div-vmin">10vmin</div>
</body>
```



```css
ol, ol li {
  font-size: 2rem
}

ul, ul li {
  font-size: 2em
}

.vw {
  font-size: 10vw
}

.vh {
  font-size: 10vh
}

.div-vw{
  width: 10vw;

}

.div-vh{
  width: 10vh;
  height: 10vh;
  background-color: rgba(0, 0, 255, 0.3);
}

  width: 10vmin;

```

## display

### 1 마진 관련

```html
<body>
  <h1>display</h1>
  <h2>block</h2>
  <p>display: block은 기본적으로 가질 수 있는 영역의 100%를 가진다.</p>
  <p>h1~h6, p, div, form, table</p>
  <div>block</div>
  <form>
    폼을 작성해주세요
  </form>
  <!-- block 가로 정렬 -->
  <div class="ml-auto">ml-auto</div>
  <div class="mr-auto">mr-auto</div>
  <div class="mx-auto">mx-auto</div>

  <!-- inline -->
  <h2>inline</h2>
  <span class="red">inline은 content 영역만큼</span>
  <span>너비를 가진다.</span>
  <input type="text">
  <input type="date">
  <a href="#">링크</a>
  <img src="#" alt="이미지">
</body>
```

```css
div {
  width: 100px;
  height: 100px;
  background-color: #dd7622;
  text-align: center;
  line-height: 100px
}

.ml-auto {
  margin-left: auto;
}

.mr-auto {
  margin-right: auto;
}

.mx-auto {
  margin: 0 auto;
}

.red {
  color: red;
}

body {
  height: 10000px
}
```

### 2 블럭관련

```html
<body>
  <!-- block-->
  <div>block</div>
  <div>block</div>
  <!-- 
    inline
    내용없이 존재 할 수 없음.
    (width, height 적용 x)
  -->
  <div class="inline">안녕하세여?</div>
  <div class="inline">내용영역이 필요합니다.</div>
  <!-- 
    inline-block
    block 속성 (width, height)
    + inline 속성 (우측 margin이 사라짐)
  -->

  <div class="inline-block">i-b</div>

    display: none; 공간자체가 없어짐.
    visibility: hidden; 공간은 유지됨.
  -->
  <div>div</div>
  <div class="none">none</div>
  <div class="hidden"></div>
  <div>div</div>
</body>
```

## Position

```html
<body>
  <div>static</div>
  <div>static</div>


  
  <!-- position이 적혀있으면 모두 static이 아니다.!!!!!! -->
  
  

  <!--
    absolute는 부모 혹은 조상요소를 기준으로 위치.
    (*가까운 조상 중에 static이 아닌 요소*)
  -->
  <!-- 
    부모가 static이어서,
    absolute1이 body를 기준으로 위치함.
  -->
  <div class="parent">
    static
    <div class="absolute">absolute1</div>
  </div>
  <!-- 
    부모가 static이 아니어서,
    absolute2가 가까운 부모를 기준으로 위치함
  -->
  <div class="parent relative">
    relative
    <div class="absolute zindex">absolute2</div>
  </div>

  <div class="parent relative-move">
      relative-move
      <div class="absolute zindex">absolute3</div>
  </div>
  <!-- 2.relative
    자기가 원래 있어야할 위치(static)을 기준으로 이동한다.
  -->
  <div class="relative-move2">relative-move</div>
  <!-- 3. fixed
    브라우저 위치에 따라 변경!
  -->
  <div class="fixed">브라우저따라</div>
</body>
```

## background

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>backgrond</title>
  <link rel="stylesheet" href="09_style.css">
</head>
<body>
  <!-- 
    block 요소는 기본적으로
    높이는 content의 영역(혹은 자식)의 높이를 가진다.
    너비는 상위 요소에 영향을 받는다.
  -->
  <section class="header">
  </section>
</body>
</html>
```

```css
.header {
  height: 300px;
  /* 이미지 경로*/
  background-image: url(bg.jpg);
  /* 이미지 크기
  -> 부족한 부분은 바둑판 형식 배치
  */
  /* background-size: 300px 300px; */
  /* background-repeat: no-repeat; */
  /* background-size: contain;
  background-repeat: no-repeat;
  background-position: center */
  background-size: cover;
  background-position: center;
}

```

1. px
   * width, height 순으로 지정    
   * 값 하나만 지정하면, width 지정 + height: auto
2. cover
   * 배경이미지 크기의 비율을 고정
   * width, height 큰 값을 배경 이미지에 맞춤
   * 보이지 않는 영역이 발생 가능함
3.   contain
   * 배경이미지크기의 비율을 고정 
   * 이미지가 보이지 않는 영역이 없도록 조정
   * 이미지 반복될 수 있음
